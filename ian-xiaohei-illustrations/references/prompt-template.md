# 十二花神 · 绛雪 Prompt Template

本文件用于规定“十二花神 · 绛雪”在生成图片时的提示词模板。

绛雪有两种主要使用形态：

1. **花神主视觉形态**：适合封面、主图、舞动日记、十二花神系列图。
2. **Q版正文插图形态**：适合文章配图、Notion、公众号、小红书、段落引导和动作包。

根据用户需求自动选择合适形态。

---

* 封面图
* 海报图
* 角色设定图

* 东方美学主题图

## 8. Prompt 使用原则

使用这些模板时，必须根据用户内容做具体化，不要机械套模板。

每次生成前至少明确：

1. 这张图是主视觉还是正文插图？
2. 使用花神形态还是 Q版形态？
3. 文章核心情绪是什么？
4. 绛雪要通过什么动作参与表达？
5. 山茶花、飘带、水袖或道具如何服务内容？
6. 是否需要中文标注？
7. 图片是否适合未来文章中复用？

最重要原则：

**绛雪不是装饰人物。她必须通过舞姿、花枝、飘带、视线、手势或道具参与文章意思的表达。**
