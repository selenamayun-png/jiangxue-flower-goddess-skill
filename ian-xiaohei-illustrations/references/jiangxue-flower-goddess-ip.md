# 十二花神 · 绛雪 IP Guide

## 角色定位

十二花神 · 绛雪，是本项目的核心角色之一。

她以“山茶”为核心意象，不只是一个古风美人形象，而是一个可持续用于文章插图、角色设定、舞动表达、情绪转译和视觉陪伴的东方角色 IP。

她的核心气质来自山茶的意象：

* 岁寒而开
* 红而不艳
* 温柔而坚定
* 端庄大气
* 安静盛放
* 柔中有骨
* 守护与陪伴
* 带一点灵动和俏皮

她既可以是：

1. **花神主视觉形态**：端庄、优雅、舞动感强，适合封面、主视觉、舞动记录。
2. **Q版卡通形态**：灵动、可爱、适合文章插图、引导、讲解、陪伴。

## 核心意象

“绛”是深红、胭脂红，有山茶花的色感。
“雪”对应山茶在寒时盛放的意象。

绛雪不是冰冷的雪，而是雪中一点温柔明亮的花色。

她像寒意之中一朵静静盛放的山茶：

* 在安静中发光
* 在柔美中自持
* 在温柔中藏着力量
* 在舞动中传递情绪与祝福

她代表的不是“茶里茶气”，而是：

* 清雅
* 稳定
* 守护
* 自持
* 温柔的生命力
* 女性的柔韧力量
* 诗意与分寸感
* 美感中的内在力量

## 花神主视觉外形设定

### 基础外形

* 东方古典女性形象
* 五官柔和、清秀、温婉
* 神情安静、自持、优雅
* 有轻微灵动感
* 身姿修长
* 仪态舒展
* 舞感强
* 动作线条流畅

### 发型与头饰

* 精致发髻
* 发间明显有山茶花装饰
* 可加入金色步摇、流苏、珠饰
* 细节精巧，但不过度繁重
* 整体要显得高雅、干净、柔美

### 服装设定

整体服装方向：

* 古风 / 古典舞服饰
* 轻纱层叠
* 水袖或长飘带
* 衣摆飘逸
* 山茶花纹样
* 柔和而有流动感

要求：

* 不要厚重宫廷感
* 不要夸张戏曲化
* 不要俗艳
* 不要过度繁复
* 不要网红写真风
* 不要现代礼服感
* 不要过度性感

推荐服装气质：

* 轻
* 柔
* 透
* 雅
* 流动
* 温润
* 有山茶盛放的感觉

## Q版卡通形态设定

### 基础风格

Q版的十二花神 · 绛雪，需要保留花神身份，但简化成更适合重复使用的角色形象。

要求：

* 头身比偏可爱
* 眼睛灵动
* 五官简洁
* 神情温柔
* 保留山茶花头饰
* 保留飘带 / 水袖 / 古风元素
* 保留柔粉色系
* 能够做很多动作
* 能用于文章正文插图

### 气质要求

* 可爱，但不幼儿化
* 灵动，但不夸张卖萌
* 有一点调皮
* 有一点俏皮
* 仍然保留优雅感
* 保留“花神”的气质，不是普通卡通女孩

### 不能变成

* 纯日漫风
* 过度二次元偶像风
* 普通古风娃娃
* 夸张表情包
* 幼稚儿童贴纸
* 只有“萌”，没有“雅”

## 用于文章插图时的原则

当十二花神 · 绛雪被用于文章插图时，必须遵守以下原则：

1. 她不只是“漂亮”，而要参与表达文章内容。
2. 她的动作、视线、花枝、飘带应服务于文章意思。
3. 每张图最好只表达一个核心情绪或一个核心概念。
4. 她可以承担“舞动解释者”或“情绪引导者”的角色。
5. 她可以根据文章内容切换为：

   * 主视觉舞姿图
   * 半身引导图
   * 小场景配图
   * Q版陪伴插图
   * 角色 + 概念物件图
6. 无论全身版还是Q版，都要保持统一角色识别度。
7. 始终保留：

   * 山茶意象
   * 优雅与端庄
   * 柔中有骨
   * 灵动但不浮夸

## GitHub 插图角色使用原则

为了让她像“小黑 illustration skill”一样可在未来文章中复用，十二花神 · 绛雪需要被视作一个“可持续插图角色”。

这意味着：

* 她不仅能出设定图
* 还要能做动作包
* 能做文章段落插图
* 能做封面角色
* 能做主题配图
* 能做概念解释图
* 能做表情和姿态变化
* 能在不同文章中重复出现，但保持一致气质

在未来生成时，优先考虑：

* 角色一致性
* 动作可读性
* 意象清晰度
* 插图适配性
* 可复用性

---

## 角色一致性检查

每次生成前后，检查以下项目：

* 是否保留山茶意象？
* 是否保留“绛雪”这一安静盛放的气质？
* 是否足够端庄、大气、优雅？
* 是否带有东方古典舞感？
* 是否有花神身份特征？
* 是否有山茶花头饰或花元素？
* 是否有柔和流动感？
* 是否避免俗艳、幼稚、夸张？
* 是否适合做文章插图？
* 是否真正参与内容表达，而不是只是造型展示？
* 如果是Q版，是否仍然保留“雅”和“花神感”？
